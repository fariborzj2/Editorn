# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2024-05-24
### Added
- **Checklist Block**: New interactive block for task lists.
- **CI/CD**: GitHub Actions workflow for automated testing and building.
- **Documentation**: Updated tutorial with Phase 15 details.

### Improved
- **Slash Menu**: Added Checklist option.
- **Meta**: Added LICENSE and CHANGELOG.md.

## [1.1.0] - 2024-05-23
### Added
- **UX Polish**: Keyboard navigation for Slash Menu.
- **Accessibility**: ARIA labels and roles.
- **Mobile**: Responsive styles.

## [1.0.0] - 2024-05-22
### Initial Release
- Core Architecture (BlockManager, PluginManager, Renderer).
- Basic Blocks: Paragraph, Header, List, Quote, Image, Divider, Code, Table, Video.
- Plugins: Slash Menu, Inline Toolbar, Autosave, Collaboration, AI Assistant (Mock).
- Framework Adapters: React, Vue.
- Unit Tests with Vitest.
