# Development Roadmap: Modern Block-Based Editor

این نقشه راه، فرآیند تبدیل ویرایشگر فعلی به یک ادیتور حرفه‌ای مبتنی بر بلوک را در قالب چندین فاز مشخص می‌کند. هر فاز شامل اهداف کلیدی و مراحل فنی مربوط به خود است.

---

## فاز ۱: معماری هسته و مدل بلوکی (Core Architecture & Block Model)
**هدف:** ایجاد بستر اصلی نرم‌افزار، جداسازی مدیریت داده‌ها از DOM، و پیاده‌سازی مفهوم اولیه «بلوک».

- [x] **طراحی کلاس‌های پایه:**
  - ساخت کلاس `EditorCore` به عنوان مدیر اصلی سیستم.
  - ساخت کلاس `BlockManager` برای کنترل لیست بلوک‌ها (افزودن، حذف، جابجایی).
- [x] **مدل داده (State Management):**
  - طراحی ساختار استاندارد JSON برای ذخیره‌سازی وضعیت بلوک‌ها (مانند `{ id, type, data }`).
- [x] **رندرینگ پایه (Base Renderer):**
  - ایجاد یک رندرکننده برای تبدیل داده‌های JSON به المان‌های DOM و برعکس.
- [x] **پیاده‌سازی بلوک پاراگراف (Paragraph Block):**
  - اولین بلوک اختصاصی که رفتار `contentEditable` را به صورت محدود مدیریت می‌کند.
  - مدیریت رویداد کلید `Enter` برای ایجاد پاراگراف جدید و تقسیم محتوا.
  - مدیریت رویداد کلید `Backspace` برای حذف بلوک خالی یا ادغام با بلوک قبلی.

---

## فاز ۲: سیستم پلاگین و نوار ابزار درون‌خطی (Plugin System & Inline Toolbar)
**هدف:** ایجاد امکان قالب‌بندی متون داخل بلوک‌ها و معماری توسعه‌پذیر برای افزودن افزونه‌ها.

- [x] **معماری افزونه‌ها (Plugin API):**
  - طراحی کلاسی برای ثبت و مدیریت افزونه‌ها.
- [x] **نوار ابزار درون‌خطی (Inline Toolbar):**
  - پیاده‌سازی یک تولبار شناور که با انتخاب متن (Selection) نمایان می‌شود.
  - کنترل Selection/Range در مرورگر برای اعمال قالب‌بندی‌ها.
- [x] **ابزارهای پایه فرمتینگ:**
  - پیاده‌سازی پلاگین‌های Bold, Italic, Underline و Link.
  - بازنویسی مکانیزم اعمال استایل به جای استفاده از `document.execCommand`.

---

## فاز ۳: توسعه بلوک‌های استاندارد (Standard Blocks Expansion)
**هدف:** اضافه کردن المان‌های متداول محتوایی که ویرایشگر را به یک ابزار کامل تبدیل می‌کند.

- [x] **بلوک تیترها (Header Block):**
  - پشتیبانی از سطوح مختلف (H1 تا H6).
- [x] **بلوک لیست‌ها (List Block):**
  - پشتیبانی از لیست‌های نقطه‌ای (Unordered) و شماره‌دار (Ordered).
- [x] **بلوک نقل‌قول (Quote Block).**
- [x] **بلوک جداکننده (Divider Block).**
- [x] **مدیریت منوی اسلش (Slash Menu):**
  - باز شدن یک منوی پاپ‌آپ هنگام تایپ `/` برای تبدیل سریع بلوک فعلی به بلوک‌های دیگر.

---

## فاز ۴: مدیریت رسانه، فایل‌ها و پیست (Media & Paste Management)
**هدف:** هندل کردن آپلود فایل‌ها، تصاویر و پاک‌سازی محتوای وارد شده از خارج ویرایشگر.

- [x] **بلوک تصویر (Image Block):**
  - آپلودر داخلی، پشتیبانی از Drag & Drop، امکان تنظیم اندازه و افزودن کپشن.
- [x] **مدیریت پیست (Paste Handler):**
  - شنود رویداد `paste`.
  - استخراج متن ساده یا تبدیل HTML خارجی به بلوک‌های استاندارد ادیتور.
  - استفاده از یک Sanitizer (مثل DOMPurify) برای پاک‌سازی محتوا.
- [x] **مدیریت جاسازی (Embed Block):**
  - برای ویدیوهای یوتیوب، آپارات و ...

---

## فاز ۵: تجربه کاربری و قابلیت‌های پیشرفته (UX & Advanced Features)
**هدف:** نزدیک کردن محصول به استانداردهای ابزارهای سطح اول جهانی.

- [ ] **تاریخچه وضعیت (Undo/Redo History):**
  - پیاده‌سازی `HistoryManager` مبتنی بر ذخیره Snapshot های JSON یا استفاده از Command Pattern.
- [ ] **جابجایی بلوک‌ها (Block Drag & Drop):**
  - امکان گرفتن و جابجا کردن بلوک‌ها در صفحه.
- [ ] **بلوک‌های پیچیده:**
  - **جداول (Table Block):** با قابلیت افزودن/حذف سطر و ستون.
  - **قطعه کد (Code Block):** همراه با Syntax Highlighting.

---

## فاز ۶: آماده‌سازی برای تولید (Production Ready & Ecosystem)
**هدف:** پایدارسازی، انتشار و ایجاد اکوسیستم برای کتابخانه.

- [ ] **پوشش تست (Testing):**
  - نوشتن تست‌های واحد (Unit Tests) با ابزارهایی مانند Jest یا Vitest برای بررسی صحت عملکرد Manager ها.
  - تست‌های Integration برای بررسی رفتار در مرورگر.
- [ ] **آداپتورهای فریم‌ورک (Framework Adapters):**
  - ایجاد کامپوننت‌های Wrapper رسمی برای React، Vue و Angular.
- [ ] **مستندسازی (Documentation):**
  - تهیه آموزش‌ها، API Reference و ایجاد یک صفحه دمو (Demo Page) حرفه‌ای.
- [ ] **انتشار پکیج:**
  - بیلد کردن به فرمت‌های UMD و ES Modules و انتشار در npm.

---

## فاز ۷: نوار ابزار ثابت (Fixed Toolbar)
**هدف:** افزودن یک نوار ابزار ثابت در بالای ویرایشگر با دسترسی سریع به تمامی امکانات ویرایشی و بلوک‌ها.

- [x] **طراحی و پیاده‌سازی `FixedToolbar`:**
  - ایجاد یک نوار ثابت در هدر ویرایشگر.
  - افزودن دکمه‌های فرمت‌بندی درون‌خطی (Bold, Italic, Underline, Link).
  - افزودن دکمه‌های درج و تبدیل بلوک‌ها (Paragraph, Header, List, Quote, Divider, Image, Embed, Table, Code).
